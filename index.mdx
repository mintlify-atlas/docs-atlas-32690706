## What's included

The sample plugin is a fully functional Obsidian plugin demonstrating the most important extension points in the Plugin API:

- A **ribbon icon** that triggers a Notice when clicked
- Three types of **command palette commands** — simple, editor-scoped, and conditional
- A **settings tab** with persistent configuration via `loadData`/`saveData`
- Safe **event listener** registration that automatically cleans up on disable
- A **Modal** UI component example
- A **status bar item** for displaying persistent text

<Note>
  Before building a new plugin, [check if one already exists](https://obsidian.md/plugins). The community has thousands of plugins and you may find a great collaborator.
</Note>

## Get started

<Steps>
  <Step title="Use this template on GitHub">
    Open [obsidian-sample-plugin](https://github.com/obsidianmd/obsidian-sample-plugin) and click **Use this template** to create your own copy.
  </Step>
  <Step title="Install dependencies">
    Clone your repo and run `npm install` to set up TypeScript, esbuild, and the Obsidian type definitions.
  </Step>
  <Step title="Develop in watch mode">
    Run `npm run dev` to compile continuously. Edit `src/main.ts` and reload Obsidian to see changes.
  </Step>
  <Step title="Install in your vault">
    Copy `main.js`, `manifest.json`, and `styles.css` to `<Vault>/.obsidian/plugins/<your-plugin-id>/`, then enable it in **Settings → Community plugins**.
  </Step>
</Steps>
